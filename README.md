# enpit-spring1
## Product Backlog
1. お客さんがリンゴを1個100円で買うことができる

   1.購入者は100円玉を1つ渡す

   2.購入者は購入した商品を受け取る

   - 100円玉しか受け付けない
2. お客さんはリンゴをまとめて買うことができる

   1.購入者は買うリンゴの個数を指定する

   2.購入者は合計金額がわかる

   3.購入者は購入した商品を受け取る

   - お釣りがあれば受け取れる
3. お客さんはドラゴンフルーツを400円で買うことができる

   1.購入フローはリンゴと同じ(他の商品と一緒に買える)

4. 会計時に小計，合計がわかる

   1.購入者は購入したものの内訳がわかる

   2.合計金額もわかる
5. お客さんはしゃけ弁当を1つ390円で買うことができる

   1.購入フローはリンゴと同じ（他の商品と一緒に買える）
